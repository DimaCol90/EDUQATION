## Управляющие структуры. FOR, IF-ELSE

Есть код:

```c
    int length = 10;
    int loc = 4;
    for(int x=1; x<=length; x++){
        if( x == loc - 1 || x == loc + 1 ){
            printf("w");
        }else if( x == loc ){
            printf("^");
        }else{
            printf("~")
        } 
    }
```

который выводит 

``` ~~w^w~~~~~```

* Добавить возможность чтения с клавиатуры **length** и **loc**
* Переделать условие так чтобы при тех же начальных данных (loc=4,length=10) результат был:

  ``` ~wW^Ww~~~~```
