## Structuri de control, for


* Avem dat un cod care afiseaza "harta" marii in simboluri
    1. **small_ship** - coordonata vaporului mic
    2. **x** - variabila care se plimba prin diapazonul 1-10
    3. **"W"** - vaporul
    4. **"~"** - apa

        ```python

        small_ship = 5
        for x in range(1,11):
            if x == small_ship:
              print( "W", end="" )
            else:
              print( "~", end="" )

        ```

        dupa rularea codului:

        ```
        ~~~~W~~~~~
        ```
