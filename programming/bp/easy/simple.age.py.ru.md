## Калькулятор возраста человека


* Предположим дан текущий год и год рождения человека, необходимо вычислить его возраст и вывести результат
    JS 

    ```javascript
    current_year  = 2019
    year_of_birth = 1990

    age = current_year - year_of_birth
    
    alert( age )
    ```  

    Python

    ```python
    current_year  = 2019
    year_of_birth = 1990

    age = current_year - year_of_birth

    print( age )
    ```
