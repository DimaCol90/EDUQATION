# Keypoints to know MAVEN

1. It's a build/management tool
2. Dependency management & transitive dependency management
3. pom.xml - passport of any module/app
4. Lifecycle:
   1. clean *
   2. validate 
   3. compile
   4. test 
   5. package 
   6. verify 
   7. install 
   8. deploy 
5. Maven has plugins
6. Maven has goals (tasks, plugin specific tasks)
