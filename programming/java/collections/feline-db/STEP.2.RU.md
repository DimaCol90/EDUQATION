## База данных с кошками

1. создать package "data"
2. Вставить interface **FelineInterface**
   1. Интерфейс должен содержать схему:
      1. public void setName()
3. Добавить класс **Cat** имплементирующий **FelineInterface**
   1. Структура класса
      1. name (String)
      2. race (String)
      3. year (int)
