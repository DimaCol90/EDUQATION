## На основе [index](./index.html) выполните следующие пункты:
1. подключите "reset.css" с любого CDN ( de ex.  [cdnjs](https://cdnjs.com))
2. подключите любой иконочный шрифт с CDN
3. подключите шрифт "Orbitron" толщиной 400,700,900 с google fonts
4. подключите файл **css/style.css**
5.Сверстайте навигационную панель, так, чтобы она адаптировалась под разные размеры экранов 3 разными способами, как показано на изображениях [result-lg.png](./result-lg.png),  [result-md.png](./result-md.png), [result-sm.png](./result-sm.png)
